# SMIMS 2021 | KI: Entscheidungsbäume
## Vorhersagen treffen
Um eine Vorhersage für eine Stunde für einen von 11 Parkplätzen in Paderbonn zu treffen, starten Sie zuerst den Script zur Daten-Aktualisierung, der Live.csv Tabelle ständig mit neuen Daten aktualisieren wird. Als nächstens, lassen sie ein Mal den Jupiter-Notebook `VorhersageTreffen.ipynb` rennen. Stellen sie einen parkplatz-Index zwischen 0 und 10. Jeztz können sie nur mit einem Klick die Parkplatzbelegung eines beliebigen Parkplatzes in der Stadt Paderbonn vorhersagen.
